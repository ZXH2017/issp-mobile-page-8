# issp-mobile-page-8
issp移动端

安装Node JS软件

转到终端或命令行。

执行以下命令来运行此应用程序。


```
$ git clone https://github.com/srinivastamada/ionic-welcome.git
$ cd ionic-welcome
$ npm install
$ ionic serve

```

ios
```
$ cordova add platform ios

$ ionic build ios

```

Android
```
$ cordova add platform android

$ ionic build android

```
